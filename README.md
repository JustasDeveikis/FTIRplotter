# FTIRplotter

A tool for visualizing and analyzing FTIR (Fourier-transform infrared) spectroscopy data.

## Required libraries
* **brukeropus**: used to read the Bruker OPUS files.
* **pandas** and **numpy**: data organization.
* **matplotlib**: generating figures.

## Features
* **Automated Plotting** Generate spectra from raw data files (OPUS format).
* **Batch processing** Handle entire directories in one go.

## Usage
1. Place your data in the 'data/' folder.
2. Run the script: 'python main.py'.
3. Check the plot.
